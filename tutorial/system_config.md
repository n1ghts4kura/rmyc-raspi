# 系统配置指南

## 一、系统安装

### 下载镜像

前往树莓派官网下载最新版 ***64-bit*** 桌面镜像

`https://www.raspberrypi.com/software/operating-systems/`

### 烧录镜像

下载镜像烧录工具 **Rufus**

`https://rufus.ie/zh/`

> 建议下载便携版

打开烧录工具

- 选择你要烧录的micro-SD卡设备

- 选择刚刚下载的镜像文件 (.img)

- 点击开始并等待完成

### 首次启动系统

账号: pi

密码: raspberrypi

自行使用外接屏幕(或VNC途径) **在桌面环境下** 进行基础系统配置

建议不要进行系统更新 (除了安全型更新)
